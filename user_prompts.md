# User Prompts History

프로젝트 '내가 왕이 될 상인가! (King Face)' 진행 과정에서 사용자가 요청한 질문 및 지시 사항 모음입니다.

## 1. 프로젝트 초기 기획 및 아키텍처 문의

> "사용자가 본인 얼굴 사진을 업로드하면 관상을 보고 조선시대 직업을 알려주는 간단한 싱글 페이지 서비스를 만들고 싶어. 서비스 이름은 '내가 왕이 될 상인가!' 어떤 아키텍처로 구성하면 좋을까? 결과값은 20가지 정도 미리 정해 놓을 거야. 동일한 얼굴로는 동일한 결과가 나오게 설정할 거야."

## 2. 아키텍처 비교 및 최적화 요청

> "아래 첨부한 내용과 너가 추천한 내용이 어떻게 다른지 설명해 줘. 그리고 더 나은 방법 또는 합쳐서 더 나은 방법이 있다면 알려 줘.
> 아이디어 자체가 밈 + 공유형이라 아키텍처는 최대한 단순 + 비용 최소 + 결과 결정성(deterministic) 이 핵심이에요."
> (참고: Serverless pHash 방식 vs Client-Side Face Analysis 방식 비교 요청)

## 3. 기술 세부 사항 확인 (해싱 방식)
