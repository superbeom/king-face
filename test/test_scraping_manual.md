# 테스트 이미지 수집 가이드 (Scraping Guide)

이 문서는 AI 모델의 일관성 검증을 위해 대량의 테스트 이미지를 자동으로 수집하는 `test/scrape-test-images.js` 스크립트의 사용법을 설명합니다.

## 1. 개요
*   **목적**: 다양한 인물의 얼굴 사진을 수집하여 `verify-consistency.js`를 통한 대규모 일관성 테스트 데이터를 확보합니다.
*   **작동 방식**: Google 이미지 검색을 사용하여 특정 연예인의 얼굴 위주 사진을 검색하고 다운로드합니다.

## 2. 사용 방법
1.  **사전 준비**: `puppeteer`가 설치되어 있어야 합니다.
    ```bash
    npm install puppeteer
    ```
2.  **스크립트 실행**:
    ```bash
    node test/scrape-test-images.js
    ```

## 3. 주요 설정 (`test/scrape-test-images.js`)
*   `CELEBRITIES`: 수집 대상 인물 리스트 (기본 남성 10명, 여성 10명).
*   `DOWNLOAD_DIR`: 이미지가 저장되는 경로 (기본: `test/images/`).
*   `IMAGES_PER_PERSON`: 인물당 수집할 이미지 개수 (기본 10개).

## 4. 구조
수집된 이미지는 다음과 같은 구조로 저장됩니다:
```text
test/
└── images/
    ├── 인물이름1/
    │   ├── 1.jpg
    │   ├── 2.jpg
    │   └── ...
    └── 인물이름2/
        └── ...
```

## 5. 주의 사항
*   **네트워크 환경**: 대량의 다운로드가 진행되므로 안정적인 인터넷 연결이 필요합니다.
*   **차단 정책**: 단시간에 너무 많은 요청을 보낼 경우 Google의 일시적인 차단이 발생할 수 있습니다.
*   **저작권**: 수집된 이미지는 **오직 테스트용**으로만 사용해야 하며, 배포 시 포함되지 않도록 주의하십시오. (`.gitignore` 확인)
